# Code minh họa lý thuyết - AdvanceTechnology

Bộ code này dùng để đính kèm khi nộp báo cáo lần 1. Nội dung  là các đoạn code ngắn minh họa cho phần cơ sở lý thuyết và công nghệ sử dụng trong đề tài.

## Nội dung

1. `01-frontend-nextjs/ProductCard.tsx`  
   Minh họa component Next.js/React hiển thị sản phẩm.

2. `02-backend-nestjs-product-service/`  
   Minh họa Product Service theo NestJS gồm Entity, DTO, Service, Controller.

3. `03-api-gateway/api-gateway.controller.ts`  
   Minh họa API Gateway nhận request từ frontend và chuyển tiếp đến product-service, cart-service, order-service.

4. `04-docker-compose/docker-compose.demo.yml`  
   Minh họa cách chạy nhiều service bằng Docker Compose.

5. `05-n8n-workflow/order-created-notification-workflow.json`  
   Minh họa workflow n8n: nhận sự kiện đơn hàng mới và gửi email thông báo.

6. `06-ai-agent-router/ai-agent-router.ts`  
   Minh họa AI Agent trung tâm điều phối yêu cầu đến các agent chuyên biệt.

7. `07-google-colab/Code_Minh_Hoa_Ly_Thuyet.ipynb`  
   Notebook có thể mở bằng Google Colab để chạy thử các ví dụ minh họa bằng Python.
